=== NPR Style Ajax Theme ===

Installation 

1) Install Wordpress E-Commerce plugin (WPEC)
2) Create product categories Track, Album, Video with slugs that are
the name in lowercase
3) Appropriate pages should be created on theme init if not there are
several page templates
for displaying each type of product
4) For each track product assign it to an album. Add the mp3 file to
the preview media upload area.
This will play in the jplayer
5) Fill out the music related meta boxes for each product to give them
proper taxonomies and meta information
